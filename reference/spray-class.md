# Class “spray”

The formal S4 class for sprays.

## Objects from the Class

Objects *can* be created by calls of the form `new("spray", ...)` but
this is not encouraged. Use functions
[`spray()`](https://robinhankin.github.io/spray/reference/spray.md) or
[`as.spray()`](https://robinhankin.github.io/spray/reference/spray.md)
instead.

## Slots

- `index`::

  Index matrix

- `value`::

  Numeric vector holding coefficients
